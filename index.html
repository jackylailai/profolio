Explicit I/O Contracts

每個 LLM call 有明確的 input / output spec 與 Pydantic schema 驗證，輸出形狀錯誤直接 fail-fast。

Prompt Versioning

Prompt / model 版本納入 audit log，可回溯哪個版本產生了哪份輸出，支援 A/B 與 rollback。

Eval Harness

Golden fixture + deterministic eval，每次 prompt 變更跑 regression report，行為可驗證、可比較。

Cost Guardrails

Token / cost / quota / privacy guardrails 內建。預算超標前阻擋，每次呼叫都有 cost audit log。

Pluggable Backend

同一份程式碼可切換 fake / Claude CLI / Anthropic API 三種 backend，離線測試零成本。

App-Owned State

狀態轉換、retry、人工 review gate 由應用程式控制，不交給 LLM 自己決定下一步要做什麼。
